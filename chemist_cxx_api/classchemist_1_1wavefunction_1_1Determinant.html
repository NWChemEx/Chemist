Detailed Description

template<typename OneParticleBasis>
class chemist::wavefunction::Determinant< OneParticleBasis >

A fermionic multi-particle wavefunction.

Template Parameters
OneParticleBasisType of the vector space spanned by the one- particle basis functions.

For indistinguishable fermions, the multi-particle wavefunction is given by the Slater determinant of the product of the occupied orbitals. This class represents that wavefunction.

Creates a Determinant from the provided orbitals and occupations.

A determinant for an -particle system is obtained by considering all permutations of a set of orbitals taken from the full set of orbitals. This ctor treats orbitals as the full set of orbitals to choose from and will build the determinant from occupied such that the orbitals[occupied[i]] is the i-th orbital in each orbital product.

Parameters
[in]occupiedThe set of orbital offsets which are occupied in THIS determinant.
[in]orbitalsThe full set of orbitals (including orbitals not occupied in *this).

The indices for the orbitals comprising *this.

An -particle wavefunction contains particles in orbitals. Those orbitals are "occupied" in the sense that a particle is in them in this determinant; however, the indices may be those of virtual orbitals if *this is a determinant representing an excited state.

Returns
A set such that the i-th element is the offset of orbital i in the determinant |\phi_0 \phi_1 \cdots \phi_i \cdots \rangle$.

Returns the set of orbitals from which the occupied orbitals are taken.

Each determinant describing particles is obtained by taking permutations of electrons in orbitals. This method returns the full set of orbitals from which the occupied orbitals are taken. Note that in this context "occupied" refers to occupied in THIS determinant, and the orbitals are not necessarily occupied in the reference determinant.

Returns
The orbital space from which the occupied orbitals are taken.
