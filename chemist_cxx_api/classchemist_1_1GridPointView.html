Detailed Description

template<typename GridPointType>
class chemist::GridPointView< GridPointType >

Aliases existing state so it can be used like a GridPoint object.

Template Parameters
GridPointTypeThe cv-qualified type of the GridPoint object *this behaves like.

The GridPointView class enables aliased data to be used as if it were stored in a GridPoint object.

inlinenoexcept

Determines if the aliased GridPoints are value equal.

Template Parameters
RHSTypeThe cv-qualified type of the GridPoint rhs is acting like.

This method compares the GridPoint aliased by *this to the GridPoint aliased by rhs. The aliased GridPoint objects are considered value equal if the:

  • weights are value equal
  • Cartesian coordinates are value equal.

Of note this comparison does not compare the addresses, i.e., it does NOT check if *this and rhs alias the same GridPoint.

Parameters
[in]rhsThe object being compared to *this.
Returns
False if *this aliases a GridPoint that is value equal to the GridPoint aliased by rhs and true otherwise.

bool chemist::GridPointView< GridPointType >::operator== ( const_grid_point_reference  rhs) const
inlinenoexcept

Determines if the aliased GridPoint is value equal to rhs.

*this is value equal to rhs if the:

  • aliased weight of *this is value equal to the weight owned by rhs
  • aliased coordinates in *this are value equal to the coordinates owned by rhs.

Of note, the addresses of the aliased state do NOT need to be the same, i.e., *this does NOT need to be aliasing rhs.

Parameters
[in]rhsThe GridPoint object to compare to.
Returns
True if the GridPoint aliased by *this is value equal to rhs and false otherwise.
