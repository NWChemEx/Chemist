Detailed Description

template<typename ChargesType>
class chemist::detail_::ChargesViewPIMPL< ChargesType >

Establishes the API for all PIMPLs used to implement ChargesView.

This is an abstract base class establishing the API the ChargesView class should rely on when interacting with its PIMPL.

Template Parameters
ChargesTypeThe cv-qualified type that *this will act like it is referencing.
