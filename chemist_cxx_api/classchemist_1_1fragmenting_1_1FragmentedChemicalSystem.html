Detailed Description

template<typename ChemicalSystemType>
class chemist::fragmenting::FragmentedChemicalSystem< ChemicalSystemType >

Stores fragments of a ChemicalSystem object.

Template Parameters
ChemicalSystemTypeThe type of the ChemicalSystem object we are fragmenting. Expected to be either ChemicalSystem or const ChemicalSystem.

This class holds a ChemicalSystem supersystem and the fragments of that object.

For design notes see https://tinyurl.com/4njpnsz5.

Fragments are formed the provided state.

This method creates an object such that the Molecule piece of the supersystem is frags.supersystem(). The supersystem is then fragmented into frags.size() fragments such that the Molecule piece of the i-th fragment is frags[i].

template<typename ChemicalSystemType >
bool chemist::fragmenting::FragmentedChemicalSystem< ChemicalSystemType >::operator== ( const FragmentedChemicalSystem< ChemicalSystemType > &  rhs) const
noexcept

Is *this value equal to rhs?

Two FragmentedChemicalSystem objects are value equal if:

  • the supersystems they fragment are value equal,
  • they contain the same number of fragments,
  • if the i-th fragment of *this is value equal to the i-th fragment of rhs.
Parameters
[in]rhsThe FragmentedChemicalSystem to compare to *this.
Returns
True if *this is value equal to rhs and false otherwise.
