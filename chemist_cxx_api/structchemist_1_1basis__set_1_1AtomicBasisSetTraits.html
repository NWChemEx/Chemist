Detailed Description

template<typename AtomicBSType>
struct chemist::basis_set::AtomicBasisSetTraits< AtomicBSType >

Propagates types associated with an AtomicBaseSet of type AtomicBSType.

This struct uses inheritance because the total set of types needed for interacting with a AtomicBasisSet is the union of the types needed for the primitives making up the AtomicBasisSet plus the types exclusive to the AtomicBasisSet.

Template Parameters
AtomicBSTypeexpected to be either an instantiation of AtomicBasisSet or AtomicBasisSetView.
