Detailed Description

template<typename ScalarType = double>
class chemist::PointCharge< ScalarType >

Models a point charge.

As the name implies, a point charge is a point with a charge associated with it. In quantum chemistry point charges are typically used to model nuclei and distant electrostatic interactions.

Template Parameters
ScalarTypethe floating-point type to use for storing the value of the charge and the charge's Cartesian coordinates. Default is double.

Determines if two point charges are different.

Two point charges are the same if they are located at the same place in space and they have the same charge. N.B. same is defined bitwise so 1.000001 != 1.00000, i.e. there is zero tolerance for difference.
