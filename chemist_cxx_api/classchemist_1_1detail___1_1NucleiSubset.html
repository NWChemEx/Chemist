Detailed Description

template<typename NucleiType>
class chemist::detail_::NucleiSubset< NucleiType >

Implements a NucleiView that is a subset of a Nuclei object.

This class is primarily envisioned as implementing NucleiView for FragmentedNuclei. In this case, we have a supersystem which is an object of type Nuclei and the FragmentedNuclei is a container of NucleiView objects which are views of the supersystem. The views can be implemented quite efficiently by storing a view of the supersystem and the indices of the nuclei in the view.

Initializes *this to alias the same Nuclei as other.

The copy ctor deep copies the indices of other, and stores a copy of the reference to the supersystem. In turn, the resulting object aliases the same supersystem (which is consistent with *this being the implementation of a view).
