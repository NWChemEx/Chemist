Detailed Description

Allows one to compare two atom instances for exact equality.

Two atom instances are defined as equal if their nuclei are identical and the number of electrons are the same. N.B that floating-point comparisons are bit-wise with zero tolerance for deviation, i.e., 1.99999999999999 != 2.00000000000000
