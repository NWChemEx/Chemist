Detailed Description

Models an ordered list of GridPoint objects.

A grid is an ordered set of grid points, where each grid point has coordinates (presently described in Cartesian coordinates) and a weight.

Note
This class does not attempt to enforce the set-like nature of the grid (i.e., it will allow duplicate grid points).
