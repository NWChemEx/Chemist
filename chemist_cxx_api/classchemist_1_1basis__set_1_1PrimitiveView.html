Detailed Description

template<typename PrimitiveType>
class chemist::basis_set::PrimitiveView< PrimitiveType >

Models a reference to a Gaussian Primitive.

This class has the same API as Primitive; however, instances of PrimitiveView do NOT own their state, they alias it.

Template Parameters
PrimitiveTypeExpected to be a, possibly cv-qualified, Primitive type.

noexcept

Is *this a view of a null primitive?

Primitives may be in a null state. Therefore it is also possible for a view of a primitive to be aliasing a null primitive. This is subtly different than a null view in that a PrimitiveView of a null primitive is still a view of a primitive, whereas a null view (if it were possible) would not be associated with a primitive.

Returns
True if *this is view of a null primitive and false otherwise.

Allows assigning Primitives to *this.

Note
This is NOT copy assignment.

As long as *this is a view of a mutable Primitive it should be possible to assign Primitives objects to the aliased Primitive. This method does NOT alias rhs, but instead sets the aliased state to a copy of the state in rhs. If you want to copy rhs, use copy assignment instead.

Template Parameters

Determines if *this is value equal to rhs.

This method value compares the center, coefficient, and exponent aliased by *this to the respective center, coefficient, and exponent aliased by rhs. If each respective comparison is true, then *this is value equal to rhs, otherwise they are different. Note that this is a value comparison, and not a determination of whether or not *this "points to" the same primitive as rhs.
