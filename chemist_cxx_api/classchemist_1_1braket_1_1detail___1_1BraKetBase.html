Detailed Description

template<typename DerivedType>
class chemist::braket::detail_::BraKetBase< DerivedType >

Code factorization for APIs and implementations common to all bra- kets.

BraKet objects come in two flavors, those that derive from TensorElement and those that derive from TensorRepresentation. The resulting BraKet operates largely the same regardless of which class it derives from. To factor out the common APIs and implementations we make both TensorElement and TensorRepresentation derive from BraKetBase via CRTP (needed so this class can get the types right for the API).

Template Parameters
DerivedTypeThe class *this is defining the API for. Assumed to be TensorElement or TensorRepresentation.

Checks for polymorphic value equality.

*this and rhs are polymorphically value equal if:

  • the most derived class of *this is the same most derived class of rhs.
  • When compared as objects of the most derived class *this and rhs compare value equal.
