Detailed Description

template<typename ChargesType>
class chemist::ChargesView< ChargesType >

Enables treating state like it's a reference to a Charges object.

Template Parameters
ChargesTypeThe cv-qualified type *this is acting like.

ChargesView objects act like an alias of a Charges object. In practice, the Charges object need not actually exist and the ChargesView object can allocate state which is not tied to a Charges object.

This ctor is a shallow copy of the aliased Charges object and a deep copy of the state in other used to alias the Charges object. In other words after this call *this will alias the same Charges object as other, but *this can be made to alias a different Charges object without affecting other.
