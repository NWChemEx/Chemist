Detailed Description

template<typename T>
class chemist::Charges< T >

Represents a container of PointCharge objects.

Conceptually the Charges class behaves like a std::vector<PointCharge<T>>; however, the backend involves a fair amount of redirection for performance reasons.

Template Parameters
TThe floating-point type used to hold each PointCharge's charge. T will also be the type of the point charge's coordinates.
