Detailed Description

Contains traits for when TiledArray is used as the backend.

These traits classes basically exist to attempt to isolate the TensorWrapper library from the guts of TA (to an extent). What really matters in all of these traits types is the variant_type member as that's what is used by the TensorWrapper library.
